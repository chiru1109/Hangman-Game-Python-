# 🎮 Hangman Game in Python

## Description

This is a simple command-line **Hangman Game** developed using Python. The game randomly selects a word from a predefined list of fruits, and the player must guess the word one letter at a time. Correct guesses reveal the matching letters, while incorrect guesses reduce the number of remaining attempts. The game ends when the player successfully guesses the entire word or runs out of six incorrect attempts.

This project demonstrates the use of Python fundamentals such as loops, conditional statements, strings, lists, user input, and the `random` module. It is a beginner-friendly project designed to strengthen programming logic and problem-solving skills.

## Features

* Random word selection using the `random` module.
* Letter-by-letter guessing.
* Displays hidden letters using underscores (`_`).
* Tracks incorrect guesses with a maximum of 6 attempts.
* Reveals correctly guessed letters.
* Displays a winning message when the player guesses the word.
* Displays the correct word if the player loses.
* Simple and interactive command-line interface.

## Technologies Used

* Python 3
* `random` module

## How to Run

1. Clone this repository.
2. Open the project in your preferred Python IDE or terminal.
3. Run the Python file:

   ```bash
   python hangman.py
   ```
4. Enter your name and start guessing one letter at a time.
5. Guess the hidden word before your six chances run out!
